# mixplugin

**mixplugin** は、PaperMC サーバー向けの便利コマンド集プラグインです。  
現在は `/sell` `/ec` `/rtp`を提供しており、今後も自由に機能を追加できる拡張性を持っています。

---

## ✨ 主な機能

### `/sell`
売却用のインベントリGUIを開きます。  
プレイヤーは **売りたいアイテムをGUIに入れて閉じるだけで自動的に売却** されます。  
売却価格は `config.yml` で手動設定できます。

### `/ec`
プレイヤーのエンダーチェストを開きます。

### `/rtp`
実行するとランダムな場所にテレポートできます
現在はオーバーワールドのみに対応しています

なぜかとてもラグくなることがあります

---

## 📦 対応環境

- **Minecraft**: PaperMC **1.21.8**
- **Java**: **21**

---

## 🔧 インストール方法

1. `mixplugin.jar` をダウンロード
2. Paper サーバーの `plugins` フォルダに配置
3. サーバーを起動すると `plugins/mixplugin/config.yml` が生成されます
4. **config.yml を手動で編集して価格やコマンドを設定してください**

---

## ⚙️ config.yml の仕様

```yaml
# 価格設定（手動編集）
prices:
  DIAMOND: 1
  IRON_INGOT: 50
  APPLE: 10
  stone: 0

# 売却時に実行されるコマンド
# %user%  → 売却したプレイヤー名
# %money% → 売却額
pay: pay %user% %money%
```

### 🔍 設定の説明

#### **prices**
- キー：Minecraft のアイテムID
    - 例：`DIAMOND`, `IRON_INGOT`, `APPLE`, `STONE` など
    - 大文字小文字はプラグイン側で処理されるため、`stone` のような小文字でも可
- 値：売却価格（整数）

> **注意**  
> 小数点未対応の場合は小数点を含む金額は正しく処理されず  
> **小数点以下は切り捨てられます（例：10.9 → 10）**。  
> そのため、`prices` は **整数値のみ** を設定してください。

#### **pay**
売却時にサーバー側で実行されるコマンド。  
経済プラグインと連携するために使用します。

利用可能な変数：

| 変数 | 内容 |
|------|------|
| `%user%` | 売却したプレイヤー名 |
| `%money%` | 売却額 |
